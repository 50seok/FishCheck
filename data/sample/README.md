# sample/ 폴더

앱 데모 및 테스트용 샘플 이미지를 여기에 보관합니다.

## 구조
```
sample/
  gwangeo/     # 광어 샘플 2~3장
  dodaree/     # 도다리 샘플 2~3장
  gajami/      # 가자미 샘플 2~3장
  bangeo/      # 방어 샘플 2~3장
  bushiri/     # 부시리 샘플 2~3장
  ureok/       # 우럭 샘플 2~3장
  gaebolak/    # 개볼락 샘플 2~3장
```

## 주의
- 생물(통 생선) 상태 이미지만
- GitHub에 포함 가능한 소용량 이미지 (각 500KB 이하 권장)
- 테스트 스크립트 (`tests/test_predict.py`) 에서 참조함
